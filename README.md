# ハイキングシミュレーター

実際の登山で必要な思考・判断・計画をゲームを通じて体験・学習できる、リアル系登山シミュレーターゲーム。

## コンセプト

「登山は山頂に立つことではなく、安全に帰ってくること」

このゲームは、実際の登山者が直面する思考プロセスや判断を忠実に再現することを目指す。
プレイヤーは登山計画の立案から出発・行動中の判断・緊急時対応まで、一連の体験を通じて
現実の山行に役立つ知識と思考パターンを身につけることができる。

### ターゲット

- 登山初心者〜中級者（実際の登山の予習・復習として）
- 登山に興味はあるが不安がある人
- 山岳安全教育の補助教材として

### プラットフォーム

- PC（Windows / macOS / Linux）
- Android

### 開発エンジン

- Godot 4
- マップデータ: 国土地理院 標高タイルAPI

---

## アイデアだし

* 国土地理院地図などから3Dマップ生成
* 水分、空腹、体力、携帯バッテリーのパラメータ
* 歩行技術レベル、登攀技術レベル
* ザック容量、重量に応じて体力消費コストが変わる
* 食料アイテムによって空腹レベルが変わる
* 水分補給地点
* 転倒、滑落、墜落などにより体力低下・状態異常発生
    * 緊急エイドキットにより回復可能、一部不可能
* 道外れなどの要素により、サバイバルフェーズへの移行あり
* 被救助されるまでのサバイバル、延命措置、ビバーク処理など
