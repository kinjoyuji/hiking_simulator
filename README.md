# hiking_simulator
ハイキングシミュレーター開発？

# アイデアだし
* 国土地理院地図、Googleマップなどから3Dマップ生成
* 水分、空腹、体力、携帯バッテリーのパラメータ
* 歩行技術レベル、登攀技術レベル
* ザック容量、重量、に応じて、体力消費コストが変わる
* 食料アイテムによって、空腹レベルが変わる
* 水分地点
* 転倒、滑落、墜落などにより、体力低下、状態異常不可
    * 緊急エイドキットにより回復可能、一部不可能
* 道外れなどの要素により、サバイバルフェーズへの移行あり
* 被救助されるまでのサバイバル、延命措置、ビバーク処理など
